# Classification Chats vs Chiens

Modèle de deep learning pour classifier des images de chats et chiens.

## Installation
```bash
pip install -r requirements.txt
```

## Préparation des données

1. Téléchargez le dataset depuis Kaggle
2. Placez `cat-and-dog.zip` à la racine
3. Exécutez : `python data.py`

## Entraînement
```bash
python train.py
```

## Prédiction
```bash
python predict.py chemin/vers/image.jpg
```
